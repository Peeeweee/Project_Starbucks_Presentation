# Starbucks-Viz: One Cup. 100,000 Stories.

A scroll-based React data storytelling presentation for CSDS 327 — Data Visualization at University of Southeastern Philippines.

## 🚀 Live Demo
[https://Peeeweee.github.io/Project-_ReLumina/](https://Peeeweee.github.io/Project-_ReLumina/)

## ☕ The Project
This presentation visualizes customer ordering patterns, demographic spending habits, and behavioral insights derived from 100,000 Starbucks transaction records. It uses a "Cracked Cup" metaphor to illustrate business challenges and their data-driven solutions.

## 🛠️ Tech Stack
- **React 18** + **Vite** + **TypeScript**
- **Framer Motion** (Cinematic scroll-based animations)
- **TailwindCSS** (High-fidelity design system)
- **React Intersection Observer** (Viewport-triggered events)
- **Google Fonts** (Playfair Display for elegance, Inter for data)
